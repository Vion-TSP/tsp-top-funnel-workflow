# TSP Top Funnel Workflow System

Three-layer marketing workflow system for The Sales People (TSP). Interactive HTML dashboards covering the full GTM engine from TOFU testing through to LinkedIn campaign execution.

## Frameworks

| # | Framework | Description | File |
|---|-----------|-------------|------|
| 1 | **TOFU Workflow Visual** | 3 parallel A/B/C tests, 5-week timeline, sub-segment workflows, 51-task checklist | [01-tofu-workflow-visual/index.html](01-tofu-workflow-visual/index.html) |
| 2 | **Unified Funnel Architecture** | Full tech stack integration map (Clay → LinkedIn → Instantly → C&S → HubSpot), 7 MOFU nurture categories, BOFU signal tracking | [02-unified-funnel-architecture/index.html](02-unified-funnel-architecture/index.html) |
| 3 | **LinkedIn Campaign Architecture** | 4 account-based campaigns, 3,099 companies, 34 content assets, full TLA + organic post drafts | [03-linkedin-campaign-architecture/index.html](03-linkedin-campaign-architecture/index.html) |

## How They Connect

```
TOFU Workflow Visual (test design)
    ↓ feeds into
Unified Funnel Architecture (tech integration)
    ↓ feeds into
LinkedIn Campaign Architecture (execution plan)
```
