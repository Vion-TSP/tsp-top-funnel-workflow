## Frameworks

### 1. TOFU Workflow Visual
3 parallel A/B/C tests (3,000 contacts), 5-week timeline, sub-segment workflows (I1/I2/I3), SDR tagging process, LinkedIn ad creative mix, measurement framework, and 51-task checklist across 6 phases.

### 2. Unified Funnel Architecture
Full tech stack integration map — Clay → LinkedIn Campaign Manager → Instantly → Connect & Sell → HubSpot CRM → Slack. 7 MOFU nurture categories (A–G, 171 existing leads), BOFU signal tracking, 15 data flows with status badges, and integration blockers.

### 3. LinkedIn Campaign Architecture
4 account-based campaigns across 3,099 companies. 34 content assets (8 TLA posts, 3 video variants, 3 stat graphics, 20 organic posts, 2 PDF playbooks). Full TLA + organic post drafts in Sully's voice, video production briefs, testimonial strategy, and 8-week posting schedule.

## How They Connect

```
TOFU Workflow Visual (test design & measurement)
    ↓ feeds into
Unified Funnel Architecture (tech integration & nurture engine)
    ↓ feeds into
LinkedIn Campaign Architecture (content & execution plan)
```
